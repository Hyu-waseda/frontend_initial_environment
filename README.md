- yarn 1.22.19
- typescript: 5.0.4
- react: 18.2.0
- react-dom: 18.2.0
- next: 13.3.1
- eslint: ^8.39.0
- prettier: ^2.8.8
- normalize.css: ^8.0.1

- CI/CD周り（GitHub Actions, trivy, dockle）
